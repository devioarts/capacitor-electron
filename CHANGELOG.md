## [0.1.1] — 2026-06-15

### Added
- Initial release of Capacitor Electron platform
- CLI commands: `add`, `copy`, `open`, `update`, `upgrade`, `kill`, `scripts`
- Support for Electron 42, Capacitor 8, Vite 8
- Plugins: App, Action Sheet, App Menu, Auto Updater, Browser, Dialog, Filesystem, Global Shortcuts, Local Notifications, Preferences, Splash Screen, Toast, Tray Menu, Window State Persistence
- Templates: `template-electron`, `template-plugin`
- IPC bridge main ↔ renderer via preload script
- Configuration via `electron-builder.js`
- Deep Linking, Content Security Policy, Icons & Assets
